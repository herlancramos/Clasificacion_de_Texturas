## 🚀 Instrucciones de Ejecución

Sigue estos pasos dentro de Visual Studio para correr la aplicación de forma correcta:

### 1. Abrir el proyecto
* Abre **Microsoft Visual Studio 2026**.
* Selecciona la opción **"Abrir un proyecto o una solución"**.
* Busca en tu equipo la ruta de la carpeta del proyecto y abre el archivo de solución independiente: `WindowsFormsApplication1.sln`.

### 2. Compilar el sistema
* En la barra de menús superior, ve a **Compilar** > **Compilar solución** (o presiona el atajo de teclado `Ctrl + Shift + B`).
* Verifica en la pestaña **Salida** (ubicada en la parte inferior) que el proceso finalice correctamente indicando `0 erróneo`.

### 3. Iniciar la aplicación
* Presiona el botón verde con el icono de play que dice **"Iniciar"** o **"Depurar"** (también puedes presionar la tecla `F5`).
* Automáticamente se desplegará la interfaz gráfica del **Clasificador de Texturas** en tu pantalla, lista para cargar las imágenes y realizar el entrenamiento por puntos.
